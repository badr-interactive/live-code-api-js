# Badr Interactive API Starterkit
This is Badr Interactive Live Code API

## Usage 
Basic command
```bash
npm run dev

npm run test

npm run build
```

## Technology
Framework:
    - ExpressJS

ORM:
    - Sequelize

Cache:
    - redis

Unit test:
    - mocha
    - chai

Logger: 
    - winston
    - morgan
    - winston-mongodb
